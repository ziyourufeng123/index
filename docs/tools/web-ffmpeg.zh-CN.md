# OmniConvert AI - 全能媒体转换器

## 功能简介

OmniConvert AI 是一个基于 WebAssembly (FFmpeg) 和 Google Gemini AI 的下一代在线媒体处理工具。它不仅支持传统的格式转换，还允许用户通过自然语言（如“把视频变成黑白并加速2倍”）来操作视频和音频，无需掌握复杂的命令行参数。所有处理均在浏览器本地完成，保护您的隐私。

## 主要特性

-   **AI 智能指令**: 集成 Gemini AI，理解您的自然语言需求并自动生成 FFmpeg 命令。
-   **本地处理**: 基于 WebAssembly 技术，文件无需上传服务器，直接在本地浏览器转换，安全快速。
-   **全能格式支持**: 支持 MP4, WebM, MP3, WAV, GIF, JPG, PNG 等多种常见媒体格式。
-   **可视化预览**: 实时预览转换后的视频或图片效果。
-   **自定义控制**: 高级用户可以直接编辑 FFmpeg 参数，实现无限可能。

## 如何使用

1.  **上传文件**: 点击上传区域或拖拽文件（视频、音频或图片）到页面中。
2.  **选择模式**:
    -   **预设工具**: 点击左侧的快捷卡片（如“静音”、“压缩”、“转GIF”等）。
    -   **AI 实验室**: 切换到 AI 标签页，输入您的需求（例如“提取音频并保存为mp3”），点击“智能分析并执行”。
3.  **配置 API Key (AI模式)**: 首次使用 AI 功能时，需输入您的 Google Gemini API Key（仅保存在本地）。
4.  **开始处理**: 确认任务后点击“开始处理”，等待转换完成。
5.  **下载结果**: 转换完成后，预览并下载您的文件。

## 注意事项

-   由于使用浏览器本地算力，处理大型视频文件（>500MB）可能会占用较多内存，建议在电脑端使用。
-   AI 功能需要科学上网环境以连接 Google Gemini API。
