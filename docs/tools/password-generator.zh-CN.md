# 随机密码生成器

生成包含大小写字母、数字和符号的强密码，支持自定义长度和字符集，保障您的账户安全。

## 功能特点

- **高度自定义**：支持设置密码长度（4-64位）。
- **多种字符集**：
  - 大写字母 (A-Z)
  - 小写字母 (a-z)
  - 数字 (0-9)
  - 特殊符号 (!@#$%^&*)
- **排除混淆字符**：可选排除易混淆的字符（如 i, l, 1, L, o, 0, O），让密码更易读。
- **强度检测**：实时显示密码强度，帮助您生成足够安全的密码。
- **一键复制**：生成后可直接点击复制按钮。
- **安全可靠**：所有计算均在本地浏览器完成，密码不会上传至服务器。

## 使用说明

1. **设置长度**：拖动滑块或查看数值，调整所需的密码长度。
2. **选择字符**：勾选需要包含的字符类型（建议至少包含三种以提高安全性）。
3. **生成密码**：点击"生成密码"按钮或刷新图标。
4. **复制使用**：点击复制图标将密码复制到剪贴板。

## 安全提示

- 建议重要账户使用 16 位以上的密码。
- 混合使用大小写字母、数字和符号可显著提高密码强度。
- 定期更换密码是良好的安全习惯。
