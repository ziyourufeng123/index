# Scientific Calculator

A powerful online scientific calculator supporting trigonometric functions, logarithms, exponents, and more.

## Features

- **Basic Arithmetic**: Addition, subtraction, multiplication, division, modulo, parentheses.
- **Advanced Functions**:
  - Trigonometry: sin, cos, tan
  - Logarithms: log (base 10), ln (natural log)
  - Exponents: x^y, e^x, square root
  - Constants: π (pi), e
- **History**: Automatically saves calculation history. Click on a history item to reuse the result.
- **Keyboard Support**: Type numbers and operators, Enter to calculate, Backspace to delete, Esc to clear.
- **Responsive Design**: Works on desktop and mobile devices.
- **Dark Mode**: Toggle between light and dark themes.

## Instructions

1. **Input Expression**: Click buttons or use your keyboard to enter a mathematical expression.
2. **Calculate**: Click "=" or press Enter.
3. **Use History**: View past calculations in the history panel below. Click an item to insert its result.
4. **Clear/Delete**:
   - `AC`: Clear all input.
   - `⌫` (Backspace): Delete the last character.

## Technology

Built with [Math.js](https://mathjs.org/) for high-precision mathematics. The interface is built with native HTML/CSS/JS for performance and simplicity.
