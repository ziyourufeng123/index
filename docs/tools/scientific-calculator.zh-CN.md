# 科学计算器

功能强大的在线科学计算器，支持三角函数、对数、指数等高级运算。

## 功能特点

- **基础运算**：支持加减乘除、取余、括号运算。
- **高级函数**：
  - 三角函数：sin, cos, tan
  - 对数函数：log (以10为底), ln (自然对数)
  - 指数函数：x^y, e^x, square root
  - 常数：π (pi), e
- **历史记录**：自动保存计算历史，点击历史记录可重新使用结果。
- **键盘支持**：支持键盘输入数字和运算符，Enter键计算，Backspace键删除，Esc键清空。
- **响应式设计**：适配桌面和移动设备。
- **暗黑模式**：支持明亮/暗黑主题切换。

## 使用说明

1. **输入表达式**：点击按钮或使用键盘输入数学表达式。
2. **执行计算**：点击 "=" 按钮或按 Enter 键获取结果。
3. **使用历史**：在下方的历史记录面板中查看过往计算，点击某条记录可将其结果填入当前表达式。
4. **清空/删除**：
   - `AC`：清空当前所有输入。
   - `⌫` (Backspace)：删除最后一个字符。

## 技术实现

本工具基于 [Math.js](https://mathjs.org/) 库开发，确保高精度的数学运算。界面采用原生 HTML/CSS/JS 构建，轻量且高效。
