# 数字转大写

## 功能简介

本工具用于将阿拉伯数字（如123.45）准确地转换为中文大写数字（如壹佰贰拾叁元肆角伍分）。它特别适用于处理财务、票据和合同等需要规范数字写法的场景，并支持标准的金额格式。

## 如何使用

1.  **输入数字**: 在输入框中填写您需要转换的阿拉伯数字金额。
2.  **点击转换**: 按下“转换”按钮。
3.  **获取结果**: 工具会自动生成符合财务规范的中文大写金额，您可以直接复制使用。

## 示例

- **输入**: `1023.5`
- **输出**: `壹仟零贰拾叁元伍角`

## 注意事项

- 请输入有效的数字，工具会自动处理小数点。
- 本工具主要为人民币金额格式设计。
