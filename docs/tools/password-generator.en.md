# Random Password Generator

Generate strong passwords with uppercase, lowercase, numbers, and symbols. Customizable length and charset to secure your accounts.

## Features

- **Highly Customizable**: Set password length (4-64 characters).
- **Multiple Charsets**:
  - Uppercase (A-Z)
  - Lowercase (a-z)
  - Numbers (0-9)
  - Symbols (!@#$%^&*)
- **Exclude Similar Characters**: Optionally exclude confusing characters (e.g., i, l, 1, L, o, 0, O) for better readability.
- **Strength Meter**: Real-time password strength indication.
- **One-Click Copy**: Easily copy the generated password to your clipboard.
- **Secure**: All generation happens locally in your browser; passwords are never sent to any server.

## Instructions

1. **Set Length**: Adjust the slider to your desired password length.
2. **Select Options**: Check the character types you want to include (using at least three types is recommended).
3. **Generate**: Click the "Generate Password" button or the refresh icon.
4. **Copy**: Click the copy icon to save the password to your clipboard.

## Security Tips

- Use passwords with at least 16 characters for important accounts.
- Mixing uppercase, lowercase, numbers, and symbols significantly increases password strength.
- Changing passwords regularly is a good security practice.
